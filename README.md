# Deepfake Detection using Spatial and Frequency Domain Analysis

A comprehensive research project investigating cross-generator generalization in deepfake detection using spatial, frequency, and hybrid feature representations.

## 📋 Overview

This project evaluates the effectiveness of different feature representations for detecting GAN-generated images, with a focus on **cross-architecture generalization**. We compare spatial-only, frequency-only, and hybrid approaches for detecting deepfakes generated by StyleGAN2 and StyleGAN3.

## 🎯 Key Findings

| Approach | Training Accuracy (StyleGAN2) | Test Accuracy (StyleGAN3) | Generalization |
|----------|-------------------------------|---------------------------|----------------|
| **Spatial-only** | 90.4% | 45.4% | ❌ Poor |
| **FFT-only** | 99.95% | 99.6% | ✅ Excellent |
| **DCT-only** | 99.5% | 99.77% | ✅ Excellent |
| **FCT-only** | 99.26% | 99.6% | ✅ Excellent |
| **Spatial+FFT** | 99% | 98% | ✅ Very Good |
| **Spatial+FCT** | 99.46% | 94.3% | ✅ Good |
| **Spatial+DCT** | 99.6% | 98% | ✅ Very Good |

**Key Insight**: Frequency-based methods significantly outperform spatial-only models in cross-generator testing, achieving up to **99.77% accuracy** on unseen StyleGAN3 data.

## 🔬 Methodologies

### 1. Spatial-Only Approach
**File**: `Spatial_only.ipynb`

Uses raw grayscale images processed through a CNN backbone.

**Architecture**:
- Conv2D(16, 4×4) → MaxPooling → Conv2D(32, 3×3) → MaxPooling → Conv2D(32, 3×3) → MaxPooling
- Dense(64) → Dense(1, sigmoid)

**Results**:
- Training (StyleGAN2): 90.4% accuracy
- Testing (StyleGAN3): 45.4% accuracy (near random guessing)
- **Conclusion**: Poor cross-architecture generalization

---

### 2. Frequency-Only Approaches

#### 2.1 FFT (Fast Fourier Transform)
**File**: `Frequency(FFT).ipynb`

Transforms images to frequency domain using FFT magnitude spectrum.

**Processing**:
```python
F = np.fft.fft2(img)
Fshift = np.fft.fftshift(F)
mag = np.abs(Fshift)
mag_log = np.log(mag + 1e-8)
```

**Results**:
- Training: 99.95% accuracy
- Testing: 99.6% accuracy
- **Best for**: Overall consistency

---

#### 2.2 DCT (Discrete Cosine Transform)
**File**: `Frequency(DCT).ipynb`

Uses DCT frequency coefficients for feature extraction.

**Processing**:
```python
freq = cv2.dct(img)
freq_log = np.log(np.abs(freq) + 1e-8)
```

**Results**:
- Training: 99.5% accuracy
- Testing: 99.77% accuracy
- **Best for**: Cross-architecture generalization (highest test accuracy)

---

#### 2.3 FCT (Frequency Channel Transform / DST)
**File**: `Frequency(FCT).ipynb`

Applies Discrete Sine Transform (DST) for frequency analysis.

**Processing**:
```python
f = dst(dst(img, type=2, norm='ortho', axis=0), type=2, norm='ortho', axis=1)
f = np.log(np.abs(f) + 1e-8)
```

**Results**:
- Training: 99.26% accuracy
- Testing: 99.6% accuracy
- **Best for**: Alternative frequency representation

---

### 3. Hybrid Approaches

#### 3.1 Spatial + FFT
**File**: `Spatial+FFT.ipynb`

Dual-branch architecture combining spatial and FFT features.

**Architecture**:
```
Spatial Branch → Flatten → Dense(64) ─┐
                                       ├→ Concatenate → Dense(64) → Output
FFT Branch → Flatten → Dense(64) ─────┘
```

**Results**:
- Training: 99% accuracy
- Testing: 98% accuracy
- **Observation**: Improved over spatial-only, but slightly lower than FFT-only

---

#### 3.2 Spatial + DCT
**File**: `Spatial+DCT.ipynb`

Combines spatial features with DCT frequency coefficients.

**Results**:
- Training: 99.6% accuracy
- Testing: 98% accuracy

---

#### 3.3 Spatial + FCT
**File**: `Spatial+FCT.ipynb`

Combines spatial features with FCT/DST representation.

**Results**:
- Training: 99.46% accuracy
- Testing: 94.3% accuracy

## 📈 Conclusions

1. **Frequency artifacts are generator-invariant**: All frequency-based methods (FFT, DCT, FCT) maintain >99% accuracy on unseen StyleGAN3 data.

2. **Spatial features generalize poorly**: The dramatic drop from 90.4% to 45.4% accuracy indicates spatial artifacts are generator-specific.

3. **DCT performs best**: Among frequency methods, DCT achieves the highest cross-architecture accuracy (99.77%).

4. **Hybrid models show intermediate performance**: While better than spatial-only, hybrid approaches don't surpass pure frequency methods, suggesting spatial features may introduce generator-specific bias.
